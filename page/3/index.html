Java 嵌套类的使用

2021-10-07

是什么

嵌套类是指被定义在另一个类内部的类，嵌套类存在的目的应该只是为了他的外围类(enclosing class)提供服务，如果将来还会出现在其他的环境中，他就该是顶层类(top-level class)。嵌套类有四种，静态成员类(static member class)，成员类(member class)，匿名类和局部类，除了第一种，其他三种都是内部类
