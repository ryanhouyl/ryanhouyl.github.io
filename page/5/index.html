Java 自动装箱&拆箱

2021-09-01

定义

自动装箱就是 Java 自动将原始数据类型转为对应的包装类对象
比如将 int 型的变量转成 Integer对象
自动拆箱反之(从 Java 1.5 开始引入)

过程

自动装箱时，编译器调用 valueOf() 将原始数据类型值转为对象；同时自动拆箱时，编译器调用类似 intValue(), doubleValue() 这类方法将对象转换成原始类型值
