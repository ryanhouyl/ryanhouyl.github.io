Java 泛型与通配符

2021-11-11

从一个问题说起

String 类是 Object 类的 subtype，但是 List<String> 却不是 List<Object>的 subtype，为什么？

最核心的原因在于**不满足里氏替换原则(LSP)**，这就引出了里氏替换原则的定义，简单来说，A 是 B 的子类型当且仅当一个有效的使用了 B 类型的程序，把其中所有的 B 类型换成 A 类型，程序的有效性依然存在，那么类型 A 就是类型 B 的子类型 (里式替换原则要求所有使用 supertype 的地方都可以透明的使用其 subtype)

Java 反射概述

2021-10-18

是什么

反射作为 Java 语言的一大特性，在底层框架实现中应用广泛，从字面上来理解，反射就类似于照镜子，是一个类自己在“照镜子”。在 Java 中我们提供了 Class 类及相关的类如 Field，Method 等来实现反射，Class 类封装了当前对象对应的类的信息，也就是 Class 类是描述我们定义的类的类。相当于在我们自己定义的类上又抽象了一层。对于每个类而言，JVM 中最多都只有一个不变的 Class 类型的对象来标示该类

2021-10-07

是什么

Java 自动装箱&拆箱

2021-09-01

定义

自动装箱就是 Java 自动将原始数据类型转为对应的包装类对象
比如将 int 型的变量转成 Integer对象
自动拆箱反之(从 Java 1.5 开始引入)

过程

自动装箱时，编译器调用 valueOf() 将原始数据类型值转为对象；同时自动拆箱时，编译器调用类似 intValue(), doubleValue() 这类方法将对象转换成原始类型值
